How I Built My Free Website

Here’s how I created my own free website from scratch using GitHub Pages and Visual Studio Code (VS Code). Follow these steps to create your own!

Step 1: Create a Folder for Your Website

Create a folder on my computer to store all the files for your website. For example: my_blog.
This folder will contain your HTML, CSS, JavaScript, and any other assets needed for your site.


Step 2: Download Visual Studio Code

Next, download Visual Studio Code (VS Code), a free code editor. It’s user-friendly and supports a wide range of programming languages and extensions.

Visual Studio Code Download


Step 3: Attach the Folder to VS Code Directory

Once VS Code is installed, open it and attach your folder, my_blog, to the VS Code workspace.
This allows you to view, edit, and manage all your project files in one place.

  • Open VS Code.
  • Click on File > Open Folder.
  • Navigate to the folder you created (e.g., my_blog) and select it.

Step 4: Attach the Folder to a GitHub Repository

GitHub allows you to store and manage your code online, which is a lifesaver should you accidentally delete your code locally or mess up all your code trying to fix one bug.

Steps to Attach the Folder to a GitHub Repository:

  • Log in to your GitHub account and create a new repository.
    (Tip: naming the repository the same name as your code folder will usually keep things from becoming confusing)
  • Clone the repository to your local machine by running the command in your terminal: 
    git clone https://github.com/your-username/your-repo-name.git
  • Move your project files into the cloned repository folder.
  • In VS Code, open the terminal and navigate to your project folder using the cd command.

Step 5: Code Away!

With everything set up, you can start writing the code for your website!
To get started, make :

  • An index.html file for the main structure of your site.
  • A style.css file for the design and layout.
  • Optional JavaScript files for interactivity.

Tip:

Download the Live Server extension in VS Code so you can view your code changes in real time. This allows you to test and preview your website locally before pushing updates to GitHub.

  • Go to the Extensions view in VS Code (click the square icon on the sidebar).
  • Search for "Live Server."
  • Click Install and follow the instructions to start a local server.

For more detailed guidance, check out this article: How to Use Live Server in Visual Studio Code.


Step 6: Push Updates to Your GitHub Repository

As you work on your website, make sure to regularly save changes and push updates to your GitHub repository. This ensures that your code is backed up and that your website stays up-to-date.

Here are the commands I use in the terminal:

  • Stage all changes: 
    git add *
  • Commit the changes with a message: 
    git commit -m "message about your update here"
  • Push the updates to GitHub: 
    git push

Step 7: Publish Your Free Website with GitHub Pages

GitHub Pages is a free service that lets you host websites directly from your GitHub repository.
To publish your website:

  • Go to your GitHub repository’s settings.
  • Scroll down to the Pages section.
  • Select the branch you want to deploy (usually main or master) and click Save.
  • GitHub will generate a link to your live website! (https://username.github.io)

More details: GitHub Pages Documentation



Happy coding!
